Փոխադրովի Wi-Fi թեժ կետ

Դուք կարող եք Ձեր սարքը վերածել փոխադրովի Wi-Fi թեժ կետի՝ իր ինտերնետային կապը համակարգիչների և այլ սարքերի հետ համօգտագործելու համար
  • Անհնար է օգտագործել սարքի ծրագրերը՝ Wi-Fi կապի միջոցով ինտերնետ մուտք գործելու համար, քանի դեռ սարքը գործում է որպես փոխադրովի Wi-Fi թեժ կետ

  • Կարող եք փոխադրովի Wi-Fi թեժ կետը կազմաձևել Wi-Fi թեժ կետի դրվածքներում

  • Լրացուցիչ տեղեկությունների համար այցելեք http://www.android.com/tether#wifi

USB տեղակապում

Ձեր համակարգիչը կարող է համօգտագործել Ձեր սարքի ինտերնետային միացումը, եթե դրանք տեղակապեք USB կաբելով
  • Տեղակապումը գործում է Windows 7, Windows Vista և Linux համակարգերով

  • Դուք չեք կարող SD քարտ միացնել Ձեր համակարգչին, երբ այն տեղակապված է

  • Լրացուցիչ տեղեկությունների համար, ներառյալ այն մասին, թե ինչպես USB տեղակապում օգտագործել այլ գործութային համակարգերի հետ, այցելեք http://www.android.com/tether#usb

Bluetooth տեղակապում

Դուք կարող եք կապել Ձեր Android սարքը Ձեր համակարգչին Bluetooth-ի միջոցով՝ Ձեր համակարգչի հետ համօգտագործելու դրա կապակցումը համացանցին
  • Bluetooth տեղակապումն աշխատում է ցանկացած համակարգչի կամ սարքի հետ, որը կարող է ինտերնետ մուտք գործել Bluetooth-ի միջոցով:

  • Լրացուցիչ տեղեկությունների համար տե՛ս http://www.android.com/tether#bluetooth
