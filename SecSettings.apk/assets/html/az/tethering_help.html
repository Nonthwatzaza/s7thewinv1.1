Portativ Wi-Fi grş nqtsi

Cihazınızın İnternet bağlantısını kompüterlər və ya digər cihazlar ilə paylaşmaq üçün onu portativ Wi-Fi giriş nöqtəsinə çevirə bilərsiniz
  • Cihaz portativ Wi-Fi giriş nöqtəsi kimi işlədildikdə Wi-Fi ilə İnternetə çıxış üçün ondan istifadə etmək mümkün deyil

  • Portativ Wi-Fi giriş nöqtəsini Wi-Fi giriş nöqtəsi parametrlərində konfiqurasiya edə bilərsiniz

  • Əlavə məlumat üçün http://www.android.com/tether#wifi daxil olun

USB bağlanma

USB kabeli ilə birləşdirsəniz, kompüteriniz telefonunuzun İnternet bağlantısını paylaşa bilər
  • Birləşdirmə Windows 7, Windows Vista və Linux ilə işləyir

  • Birləşdirmə olduqda SD kartı kompüterinizə qoşa bilməzsiniz

  • USB birləşdirilməsini digər əməliyyat sistemləri ilə necə istifadə etməyə dair məlumat da daxil olmaqla əlavə məlumatlar üçün http://www.android.com/tether#usb

Bluetooth bağlanması

Cihazınızın İnternet bağlantısını kompüterinizlə paylaşmaq üçün Android cihazınızı Bluetooth bağlantısı ilə kompüterə qoşa bilərsiniz
  • Bluetooth bağlanması Bluetooth vasitəsilə İnternet bağlantısı əldə edə bilən istənilən kompüterlə və ya başqa cihazla işləyir
