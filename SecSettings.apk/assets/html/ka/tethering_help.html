პორტ. Wi-Fi ცხელი წერტილი

შეგიძლიათ გამოიყენოთ თქვენი აპარატი, როგორც პორტატული Wi-Fi ცხელი წერტილი და მისი ინტერნეტ-კავშირი გააზიაროთ კომპიუტერთან ან სხვა მოწყობილობასთან
  • შეუძლებელია აპარატის პროგრამების გამოყენება ინტერნეტის ჩასართავად Wi-Fi კავშირით, როდესაც აპარატი მუშაობს, როგორც პორტატული Wi-Fi ცხელი წერტილი

  • შეგიძლიათ აირჩიოთ პორტატული Wi-Fi ცხელი წერტილის კონფიგურაციის პარამეტრები, Wi-Fi ცხელი წერტილის პარამეტრებში

  • დამატებითი ინფორმაციისთვის იხ.: http://www.android.com/tether#wifi

ნოუთბუქში მობილური ინტერნეტი USB-თი

თუ კომპიუტერს USB სადენით დაუკავშირებთ აპარატს, კომპიუტერი შეძლებს გაიზიაროს ტელეფონის ინტერნეტ-კავშირი
  • კომპიუტერში მობილური ინტერნეტ-კავშირი მუშაობს Windows 7, Windows Vista და Linux სისტემებში

  • SD ბარათს ვერ დააკავშირებთ კომპიუტერთან, როდესაც ჩართულია მობილური ინტერნეტ-კავშირი

  • დამატებითი ინფორმაციისთვის, მათ შორის სხვა ოპერაციულ სისტემებში USB კავშირით მობილური ინტერნეტის გამოყენების შესახებ ეწვიეთ საიტს http://www.android.com/tether#usb

მობილური ინტერნეტის გაზიარება Bluetooth-ით

შეგიძლიათ თქვენი Android აპარატი ინტერნეტი Bluetooth-ით დაუკავშიროთ კომპიუტერს და გააზიაროთ მასთან მობილური ინტერნეტი
  • მობილური ინტერნეტი Bluetooth-ით მუშაობს ნებისმიერ კომპიუტერში ან მოწყობილობაში, რომელსაც შეუძლია დაუკავშირდეს ინტერნეტს Bluetooth-ით

  • დამატებითი ინფორმაციისთვის იხილეთ http://www.android.com/tether#bluetooth
